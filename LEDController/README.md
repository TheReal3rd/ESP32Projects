# ESP32 LED Controller
This is a basic project that i've completed within a week.

An ESP32 controlling a set of ws2812 with Master and slave node fuctionality. Using get and json responses to have options and modes set and then json used to provide error and status updates to the user.
An external program is intended to be used to control these which hasnt been created yet.

Used to light up underneath a set of desks at my place of work.
